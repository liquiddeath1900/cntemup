# US Bottle Deposit Rates by State

*Last updated: 2026-02-15*

## States WITH Bottle Deposit Laws (10 states)

| State | Standard Rate | Special Rates | Notes |
|-------|---------------|---------------|-------|
| California | 5¢ | 10¢ (24oz+), 25¢ (boxed wine/pouches) | CRV program, broadest coverage |
| Connecticut | 10¢ | - | Increased from 5¢ in Jan 2024 |
| Hawaii | 5¢ | +1¢ non-refundable fee | HI-5 program |
| Iowa | 5¢ | - | Carbonated + alcohol only |
| Maine | 5¢ | 15¢ (wine/liquor) | Most comprehensive bottle bill |
| Massachusetts | 5¢ | - | Carbonated only, ~43% redemption |
| Michigan | 10¢ | - | Highest single rate, 97% return rate |
| New York | 5¢ | - | Carbonated + water, <1 gallon |
| Oregon | 10¢ | - | Includes canned wine, 87% return rate |
| Vermont | 5¢ | 15¢ (spirits) | First bottle bill state (1953) |
